# Enigma

Enigma is a simple Ruby library created to explain Enigma machines works. It simulates the rotors, reflector and plugboard that compose Enigma machines and provides a simple CLI for experimenting.

## Installation

    git clone ...
    cd enigma

## Usage

    bundle exec enigma
    ENIGMA
    CKASLQ

    bundle exec enigma
    CKASLQ
    ENIGMA
